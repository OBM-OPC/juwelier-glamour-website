## Technische Details

### File-Struktur
```
juwelier-glamour-website/
├── index.html          # Goldankauf (Startseite)
├── trauringe.html      # Trauringe & Verlobungsringe
├── kreuzschmiede.html  # Kreuzschmiede Ungemacher
├── ueber-uns.html      # Über uns / Team
├── kontakt.html        # Kontakt + Impressum + Datenschutz
├── css/
│   └── style.css       # ~650 Zeilen, responsiv
├── js/
│   └── main.js         # Mobile Menu, Form-Validierung, Lightbox
└── netlify.toml        # Deployment-Config
```

### Features
- **Mobile-First:** Responsive Breakpoints bei 768px und 1024px
- **Navigation:** Sticky Header mit Scroll-Effekt
- **Animations:** CSS-Animationen für Interaktionen
- **Accessibility:** ARIA-Labels, Screenreader-optimiert
- **Performance:** Optimierte Bilder (Unsplash CDN), Lazy Loading
